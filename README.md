# What's Your Coffee Personality? ☕

An interactive quiz that matches you to one of 4 coffee personalities — built for **Basecamp Coffee's** loyalty program.

**Live:** [quiz-project-lake-six.vercel.app](https://quiz-project-lake-six.vercel.app/)

## The Quiz

Answer 6 fun questions about your lifestyle, taste, and vibe — then get a personalized coffee personality with a percentage breakdown of all 4 types.

### Personality Types

| Personality | Coffee | Tagline |
|---|---|---|
| Sweet Enthusiast | Caramel Latte | "Life's too short for bitter" |
| Zen Minimalist | Black Coffee, Single Origin | "Simple. Clean. Perfect." |
| Social Butterfly | Cappuccino | "Coffee is better with company" |
| Health Nut | Oat Milk Americano | "Wellness in every sip" |

### Features

- 6 emoji-rich questions with smooth slide transitions
- Percentage-based results with all 4 personalities
- Coffee photos for each personality
- Share button (native share on mobile, clipboard on desktop)
- Fully responsive for mobile
- Warm, cozy visual style with earth tones

## Tech Stack

- **Next.js 16** + **React 19** + **TypeScript**
- **Tailwind CSS 4**
- **Playfair Display** + **Source Sans 3** (Google Fonts)
- Deployed on **Vercel**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the quiz.

## Part of Basecamp Coffee

This quiz is the centerpiece of Basecamp Coffee's loyalty program redesign — replacing a generic points system with personality-driven coffee discovery.
